# TransMed

TransMed é um projeto desenvolvido para calcular e organizar os gastos mensais com transporte, oferecendo ao usuário uma visão clara de quanto é gasto em diferentes meios de locomoção, como:

* Ônibus
* Carro
* Trem / Metrô
* Uber e outros aplicativos de transporte

## Objetivo

O objetivo do TransMed é facilitar o controle financeiro, permitindo que o usuário acompanhe e compare seus gastos com transporte ao longo do mês, auxiliando na tomada de decisões para otimização de custos.

## Funcionalidades previstas

* Cálculo automático dos gastos mensais
* Cadastro e edição de diferentes tipos de transporte
* Comparação entre categorias de gastos
* Interface simples, objetiva e responsiva

## Tecnologias utilizadas

* React
* TypeScript
* Tailwind CSS

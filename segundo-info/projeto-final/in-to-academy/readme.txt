In to academy - Plataforma de aprendizado de educação do ensino fundamental

Visão geral: O projeto em questão é uma plataforma de educação do ensino fundamental utilizando apostilas, videoaulas e atividades como quizzes. As aulas são divididas por matérias do ensino fundamental, como matemática, português, geografia, etc. O projeto trata sobre crianças e adultos que não possuem uma educação básica e de qualidade.


Objetivos:

1. Levar acesso à educação básica, de qualidade e divertida;

2. Facilitar o acesso;

Ferramentas utilizadas: O site foi construído apenas para a visualização (front), utilizando HTML, CSS e JS.


Os arquivos foram divididos em HTML: index.html para a página principal e login.HTML para a página de entrada ou criação de conta do usuário. CSS: com o style.CSS, onde está toda a estilização do site, desde paleta de cores a algumas animações, como na tela de login. Javascript: login.js, aonde tem o "start" para iniciar a animação do container do login; script.js, aonde o código para mudar do modo claro para o noturno está armazenado; searchbar.js, script que entrega vida à barra de pesquisa, na qual ela tem uma certa funcionalidade.

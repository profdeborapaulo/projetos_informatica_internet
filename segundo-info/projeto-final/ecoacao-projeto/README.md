EcoAção – Denúncias Ambientais

O EcoAção é um site simples onde moradores podem registrar denúncias ambientais, enviando descrição, localização e foto.
As denúncias são salvas em um arquivo db.json e exibidas na página de forma organizada.

Como rodar o projeto
1. Instalar dependências
No terminal, dentro da pasta do projeto:
npm install express multer cors

2. Iniciar o servidor
node server.js

3. Acessar no navegador
http://localhost:3000

Introdução

Este guia foi criado para jogadores que desejam dominar o jogo, seja para alcançar o tempo mais rápido em uma speedrun ou completar tudo no modo 100%. Aqui você encontrará estratégias, rotas otimizadas e recursos usados por speedrunners e completistas ao redor do mundo.

As informações podem ser aplicadas a diversos jogos da franquia Kirby

Any% Speedrun guia

    SS = Sun Stone. Você precisa coletar uma certa quantidade em cada mundo para progredir.

    A habilidade abaixo de cada número de fase é a que você usará durante toda a fase, a menos que seja indicado o contrário. As setas indicam que você deve trocar para a habilidade para a qual elas apontam.

    O tempo começa ao selecionar o modo história em um novo arquivo.

Dicas gerais:

  • Use um checklist ou mapa interativo para não esquecer itens escondidos.
  • Combine exploração com coleta eficiente de recursos.
  • Planeje a ordem das missões para reduzir viagens desnecessárias.

Dicas Avançadas

  • Assista a runs de recordistas no Speedrun.com para aprender técnicas e rotas atualizadas.
  • Treine segmentos específicos (splits) isoladamente antes de tentar uma run completa.
  • Grave suas tentativas e revise os erros para melhorar consistência.
  • Use ferramentas como LiveSplit para monitorar seu desempenho em tempo real.

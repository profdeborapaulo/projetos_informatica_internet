# Projeto Web: Manual de Segurança da Informação

Este projeto foi desenvolvido como parte da disciplina de Programação Web na ETEC Antônio Furlan,
 sob orientação da professora Debora. O site tem como objetivo apresentar, de forma clara e interativa, 
 os pilares da Segurança da Informação.

##Sobre o Projeto

O site é composto por páginas temáticas que explicam os conceitos de Confidencialidade, Integridade, Disponibilidade, 
Autenticidade e Não-repúdio. Cada seção traz conteúdo didático, visual e interativo para facilitar o entendimento dos usuários.

##  Base Conceitual

O conteúdo do site foi baseado no relatório técnico “Os pilares da Segurança da Informação”, que aborda:

- Confidencialidade  
- Integridade  
- Disponibilidade  
- Autenticidade  

## Como Executar

1. Clone o repositório privado da professora Debora no GitHub.  
2. Abra a pasta `MANUAL` no Visual Studio Code.  
3. Execute o arquivo `inicio.html` em seu navegador.

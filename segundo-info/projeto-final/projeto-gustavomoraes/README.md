======================== MeEnsina ========================
=== Introdução

O MeEnsina é um projeto educacional desenvolvido para facilitar o aprendizado por meio de tutoriais práticos, vídeos explicativos e compartilhamento de conhecimento entre usuários.
O site funciona como uma plataforma de ensino colaborativo, onde qualquer pessoa pode buscar, assistir e enviar tutoriais sobre diversos temas — desde tecnologia até tarefas do dia a dia.

O projeto foi criado como uma iniciativa acadêmica e prática, com foco em usabilidade, acessibilidade e aprendizado intuitivo, servindo também como um protótipo funcional para futuras expansões com backend e login real.

=== Objetivos do Projeto

- Tornar o acesso ao conhecimento simples, rápido e gratuito.

- Estimular o compartilhamento de tutoriais entre usuários.

- Demonstrar boas práticas em desenvolvimento front-end.

- Servir como base para projetos educacionais e de ensino online.

=== Público-Alvo

- Estudantes que buscam aprender de forma independente.

- Pessoas que desejam ensinar e compartilhar seus conhecimentos.

- Instituições e professores que promovem o uso de tecnologias educacionais.

=== Principais Funcionalidades

- Busca de tutoriais por categoria, nome ou palavra-chave.

- Visualização de vídeos diretamente do YouTube, com embed automático.

- Envio de tutoriais por formulário (título, autor, passos e vídeo).

- Comentários nos tutoriais, estimulando a troca de conhecimento.

- Filtro por categoria com exibição dinâmica via URL.

- Layout responsivo e acessível em qualquer dispositivo.

=== Conceito de Design

A identidade visual do MeEnsina é simples, leve e direta, refletindo o propósito educativo do projeto.
A interface foi pensada para que o usuário encontre e consuma conhecimento com rapidez e clareza.

- Layout responsivo (Bootstrap 5).

- Ícones ilustrativos e tipografia legível.

- Paleta de cores: 
  #053E61;
  #3F88C5;
  #fff;


=== Fluxo Principal do Usuário

- Acessar o site e buscar tutoriais.

- Visualizar tutoriais por categoria ou palavra-chave.

- Assistir vídeos embutidos (YouTube).

- Enviar novos tutoriais com título, descrição, passos e vídeo.

- Interagir com comentários na seção de feedback.

=== Visão Técnica (Resumo)

O MeEnsina foi desenvolvido utilizando apenas tecnologias front-end e uma API externa de mock (MockAPI.io), garantindo praticidade na demonstração das funcionalidades.

Tecnologias Utilizadas

- HTML5 + CSS3: estrutura e estilo visual.

- Bootstrap 5: layout responsivo.

- JavaScript (ES6): lógica e interatividade.

- MockAPI: armazenamento e recuperação de dados via API.

=== Estrutura Conceitual

O código é organizado de forma modular, com cada parte responsável por uma função específica (busca, envio, exibição e comentários).

Estrutura de Pastas

/projeto
  ├── index.html          # Página inicial
  ├── index2.html         # Versão logada
  ├── perfil.html         # Página de perfil do usuário
  ├── contato.html        # Página de contato
  ├── login.html          # Tela de login
  ├── cadastro.html       # Tela de cadastro
  ├── style.css           # Estilo principal do site
  └── script.js           # Funções gerais e integração com a API

/img
  └── Logos, ícones e imagens ilustrativas


=== Páginas e Responsabilidades

- index.html: Página inicial com busca e exibição de tutoriais.
- index2.html: Versão acessível após login.
- perfil.html: Mostra informações do usuário.
- contato.html:	Canal para dúvidas e suporte.
- login.html: Entrada no sistema (simulada).
- cadastro.html: Registro de novos usuários.
- categorias.html: Filtragem dos tutorias por categorias.  


=== Propostas Futuras

O MeEnsina está em constante evolução. Entre as possíveis melhorias:

- Sistema de login com autenticação real.

- Perfis personalizados e ranking de tutores.

- Área administrativa para validação de tutoriais.

- Implementação de banco de dados real.

- Chat e fóruns de dúvidas entre usuários.

- Sistema de contratação de serviços, permitindo que usuários encontrem e contratem profissionais de forma segura e integrada à plataforma.


=== Considerações Finais

O MeEnsina nasceu da vontade de tornar o conhecimento mais acessível e de mostrar como a tecnologia pode ser uma ponte para o aprendizado colaborativo.
Mesmo sendo um protótipo, ele representa o potencial de unir simplicidade, design e educação digital em um só lugar.

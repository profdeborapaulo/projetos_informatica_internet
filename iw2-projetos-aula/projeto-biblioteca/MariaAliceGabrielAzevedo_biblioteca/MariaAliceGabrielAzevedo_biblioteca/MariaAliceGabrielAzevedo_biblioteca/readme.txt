Biblioteca Virtual 15/09/2025
-----------------------------------------------------------------------------------------------------------------------------------

Este projeto nasceu da ideia de criar uma página simples, bonita, interativa  com a temática infanto-juvenil para apresentar livros de 
forma diferente.

-------------------------------------------------------------------------------------------------------------------------------------------

Funcionalidades: 

- Um cabeçalho estilizado que dá as boas-vindas;  
- Uma estante organizada, com livros principais dispostos em fileiras; 
- Efeito de frente e verso nos livros (ocorre ao passar o mouse em cima);  
- Uma seção Sobre Nós, contando um pouco mais sobre o projeto;
- Botões e interações que mudam de cor e brilho ao serem clicados ou destacados; 
- Um rodapé simples, para fechar a página.  

--------------------------------------------------------------------------------------------------------------------------------------


Organização do projeto:

 - index.html = página principal;

 - livro1.html, livro2.html, livro3.html, livro4.html, livro5.html e livro6.html = páginas dos livros com sinopse e áudio;

 - style.css = arquivo responsável por estilizar header, nav, footer... Mas principalmente a tela página inicial;

 - livro1.css = arquivo responsável por estilzar a página de todos os livros;

 - pasta img = pasta com as imagens usadas;

 -pasta audios = pasta com os áudios usados;

 README.txt = arquivo com as informações base sobre o site.

----------------------------------------------------------------------------------------------------------------------------

Execução Local:

 - Baixe a pasta;

 - Clique com o botão direito emcima dela;

 - Selecione a opção "extrair tudo";

 -Baixe a extensão "preview on web sever" no Vs Code;

 - Clique com o botão direito na página "index.html";

 - Aperte Ctrl + shift + L.

 Agora você poderá visualizar a Biblioteca Virtual localmente no seu navegador!

O projeto funciona a base de HTML, CSS e Javascript puro, sem nenhum tipo de framework, com a leitura das informações dos livros funcionando por um comando modal que abre um Pop-up do respectivo livro clicado, aparecendo suas informações e um áudio da sinopse para pessoas deficientes visuais/disléxicos.

O projeto é estruturado por HTML, CSS e Javascript, com o conteúdo dele sendo devidamente separado entre pastas com as imagens e os áudios.

Para a execução local do projeto, basta somente baixar a pasta do projeto, abri-la, e executar o arquivo HTML com o nome de index.

Biblioteca Fitipaldi

A Biblioteca Fitipaldi é um project fictício  de uma biblioteca virtual, que foi criado para apresentar aos leitores o universo dos mahwas (quadrinhos coreanos). O site destaca obras populares e oferece informações detalhadassobre cada título.

Funcionalidades

- Página inicial intuitiva: Livros em destaque organizados em carrosséis;

- Modal de detalhes: Botão "Saiba Mais" abre um pop-up com informações adcionais sobre o livro selecionado.

- Descrição com áudio: Ouça a sinopse do Livro.

- Design responsivo: Visualização adaptada para computadores, tablets e celulares.

- Acessibilidade: Suporte a áudio para facilitar o acesso à sinopse.

Estrutura

biblioteca-atividade

- index.html  (Página principal do site)
- style.css   (Estilos visuais do site)
- script.js   (Funcionalidade e interatividade)
- img/        (Imagens das capas e livros) 
- audio/      (Áudios das sinopses para acessibilidade)
README.txt    (Arquivo contendo as informações do site e suas Funcionalidades)


Como Executar Localmente

1. Pré-requisitos 

-Navegador web (Chrome, Firefox, Edge);
- Não é necessário instalar nada, pois o projetoi é totalmente front-end.

2. Passos

- Baixe ou clone o repositório para seu computador;
- Abra a pasta 'Biblioteca-atividade/biblioteca';
- Clique duas vezes no arquivo 'index.html' para abrir no navegador.

3. Navegação

- Explore os carrosséis de livros;
- Clique em "Saiba Mais" para ver detalhes de cada obra;
- Utilize o botão de áudio para ouvir a descrição.

Observações

Esse projeto foi feito apenas para fins educacionais e não comercializa nenhum conteúdo;
As imagens, áudios e as descrições dos livros são fictícias ou utilizadas apenas como exemplo.
